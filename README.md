# Блокировщик работы Мессенджера Макса

Скрипт для комплексной блокировки и сетевой изоляции Мессенджера MAX на операционной системе Windows.

## Возможности

* **Модульное меню:** Позволяет гибко выбирать тип ограничений прямо в процессе работы.
* **Отключение доступа к сети:** Полная блокировка IP-адресов, подсетей VK и DNS-изоляция доменов через файл hosts.
* **Запрет на запуск EXE:** Жесткое ограничение на уровне реестра (Image File Execution Options) — приложение физически не запустится, даже если оно обновлено или перенесено.
* **Принудительное завершение:** Автоматически закрывает все фоновые процессы мессенджера и задачи в планировщике Windows.
* **Функция отката изменений:** Полное удаление всех созданных правил и восстановление работоспособности системы в один клик.

## Инструкция по использованию

1. Скачайте файл `Max_block.bat` на свой компьютер.
2. Нажмите правую кнопку мыши на файле и выберите **"Запуск от имени администратора"**.
3. Следуйте интерактивным инструкциям и выберите нужный пункт (1-4) в открывшемся окне командной строки.
