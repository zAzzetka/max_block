# Блокировщик работы Мессенджера MAX

Скрипт для комплексной блокировки, сетевой изоляции и контроля автозапуска Мессенджера MAX.

## Возможности

* **Модульное интерактивное меню:** Позволяет гибко управлять всеми функциями защиты.
* **Отключение доступа к сети:** Блокировка IP-адресов, подсетей VK, WebRTC-портов и DNS-изоляция доменов через файл hosts.
* **Запрет на запуск EXE:** Ограничение на уровне реестра (Image File Execution Options) — приложение не откроется, даже если обновится.
* **Управление автозапуском:** Встроенная функция добавления скрипта в Планировщик задач Windows для автоматического фонового применения правил при старте системы.
* **Функция полного отката:** Удаление всех созданных правил, веток реестра и задач автозапуска в один клик.

## Инструкция по использованию

1. Скачайте файл `Max_block.bat` на свой компьютер.
2. Нажмите правой кнопкой мыши на скачанный файл и выберите **"Запуск от имени администратора"**.
3. В открывшемся меню введите цифру нужного действия и нажмите **Enter**:
   * **Выбор `1`** — Перекроет мессенджеру доступ в интернет и заблокирует его сайт.
   * **Выбор `2`** — Полностью запретит запуск самой программы `max.exe` на компьютере.
   * **Выбор `3`** — Добавит этот скрипт в автозагрузку Windows (блокировка будет активироваться сама при включении ПК).
   * **Выбор `4`** — Полностью удалит скрипт из автозагрузки системы.
   * **Выбор `5`** — Сбросит все ограничения и вернет систему в исходное рабочее состояние.
   * **Выбор `6`** — Закроет утилиту.
